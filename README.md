# My Profile App

A simple profile display mobile app built with **React Native + Expo**.  
Great for practicing UI design and understanding basic component structure.

---

## 📦 Technologies Used

- React Native (via [Expo](https://expo.dev))
- Expo CLI
- React Navigation
- React Native Vector Icons
- JavaScript (ES6+)

---

## 🚀 Getting Started

### 1. Clone this repository
`git clone https://github.com/Onpreeya-Jantakote/profile-app.git`
`cd profile-app`

### 2. Install dependencies
`npm install`
  
### Run the App
`npx expo start`
